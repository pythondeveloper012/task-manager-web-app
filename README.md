# 📝 Todo Web Application

## 📌 Project Overview
This is a web-based Todo application developed using Python and Django. The purpose of this project is to build a practical task management system while gaining hands-on experience with backend development and database integration.

The application allows users to create, update, and delete tasks in an organized way. It demonstrates the implementation of CRUD operations, Django models, views, templates, and basic project structure.

## 🚀 Features
- Create new tasks
- Update existing tasks
- Delete tasks
- Mark tasks as completed
- Organized task management
- Backend integration using Django

## 🛠 Technologies Used
- Python  
- Django  
- HTML  
- CSS  
- SQLite (Default Django Database)

## 🎯 Learning Objectives
This project was built to strengthen my understanding of:
- Django framework structure
- CRUD functionality
- Database models and migrations
- Connecting frontend with backend
- Building dynamic web applications

## 📂 How to Run the Project
1. Clone the repository
2. Navigate to the project directory
3. Install required dependencies  
   `pip install -r requirements.txt`
4. Run migrations  
   `python manage.py migrate`
5. Start the development server  
   `python manage.py runserver`
6. Open the local server link in your browser
